# Experiment 5
## Buzzer
### __ITEMS NEEDED:-__
* Arduino UNO
* A Breadboard
* Male to male jumper wires (x2)
* A Buzzer
* USB cable to connect the arduino


### Code:

 ```
void setup() 
{ 
  pinMode(8,OUTPUT);
} 
void loop() 
{
digitalWrite(8, HIGH); 
}

```
### _Output:_
A buzzer starts producing sound
